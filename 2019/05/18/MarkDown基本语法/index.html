MarkDown基本语法

一、标题

一级标题#
二级标题##
三级标题###
四级标题####
五级标题#####
六级标题######

二、字体

  • 加粗 
    将要加粗的字体的左右分别用两个*包起来

  • 斜体

    将倾斜的字体的左右分别用一个*包起来

  • 斜体加粗

    将字体左右分别用三个*包起来

  • 删除线

    将字左右分别用两个~包起来

    ###三、引用

    这是引用的内容,用一个’>’加在句子最前

    ###四、分割线
    三个或者以上的—、***
